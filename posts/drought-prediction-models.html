Drought Prediction Models with Climate Data

Posted: Apr 2024

This project explored supervised and unsupervised models for drought prediction using NOAA climate data, USGS groundwater datasets, and USCRN soil moisture/temperature. Models compared included PCA + Random Forest and Support Vector Machines (SVM).

The analysis showed that Random Forest provided strong baseline accuracy, while PCA preprocessing helped improve interpretability and dimensionality reduction. SVM performed well in separating drought vs. non-drought scenarios, but required careful parameter tuning to avoid overfitting.

Tools used: Python, scikit-learn, PCA, Random Forest, SVM.
